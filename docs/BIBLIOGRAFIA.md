## Notas sobre el Formato APA

- Las referencias están ordenadas alfabéticamente por apellido del autor o nombre de la organización.
- Se utiliza el formato APA 7ª edición (2020).
- Para software y frameworks, se indica la versión entre corchetes después del título.
- Las URL se incluyen sin "Recuperado de" según APA 7.
- Las fechas de acceso no son necesarias para contenido estable según APA 7.
