# Actores del Sistema

Lista de actores (ejemplos):

- Admin: gestiona usuarios, materias, semestres, genera horarios.
- Docente: consulta su horario, edita disponibilidad.
- Operador (Secretaría): asigna aulas, gestiona grupos.
- Visitante: consulta horarios públicos.

Para cada actor anotar responsabilidades y privilegios.
