# Anexos

Carpeta para: SQL scripts, capturas de pantalla, logs, seeders relevantes y otros artefactos.

Ejemplos recomendados:
- `anexos/schema.sql` (DDL exportado)
- `anexos/seed_sample.sql` (ejemplos de INSERT)
- `anexos/logs/` (logs de despliegue relevantes)
- `anexos/screenshots/` (capturas de pantallas del sistema)
